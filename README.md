# OneSquare

> Just another places application.

## Web

### Requirements

* Git
* Node.js
* Phonegap

### Installing

```bash
$ git clone https://github.com/INTEC-2015-Mobile-Group-6/onesquare
$ cd onesquare/src/onesquarepg
$ npm install
$ npm start
```

### Building
```bash
$ cordova platform add android
$ cd onesquare/src/onesquarepg
$ cordova build android
```
